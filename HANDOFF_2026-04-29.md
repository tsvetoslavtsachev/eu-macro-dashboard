## Anti-patterns (avoid)

1. **Paper-over contradictions** — frame като tension с lag, не като "transmission broken"
2. **Title overstatement** — title не може да каже нещо, което audit reject-на
3. **Adjective leakage** — числата носят signal-а; "едва", "mirage" са bias
4. **All-of-the-above reasoning** — rank alternatives by likelihood
5. **Forward prediction без base rate** — explicit historical %
6. **Circular hypothesis rejection** — alternatives test същите data с different framings
7. **Pure cache read без TTL check** — auto-refresh kicks in (Phase 7)
8. **Default 3 симетрични тези** — false equivalence

## Технически gotchas (lessons learned)

1. **Eurostat geo код варира по dataset:**
   - `EA21` за `une_rt_m`, `sts_inpr_m`, `sts_trtu_m`, `lfsi_emp_q`, `teibs*`
   - `EA` (auto-shifting) за `prc_hicp_manr`
   - `EU27_2020` за full EU
   - Винаги test fetch без geo filter първо за да видиш валидни кодове

2. **Eurostat 413 Payload Too Large** при липсващи filters — добави `unit=`, `coicop=`, `nace_r2=`, etc. за намаляване

3. **Eurostat period formats: `1983-01` (с тире) vs `1983M01` (без)** — нашият parser handle-ва и двете (8 паралелни regex patterns)

4. **ECB SDMX 7-dimensional keys:** SPF е `FREQ.REF_AREA.FCT_TOPIC.FCT_BREAKDOWN.FCT_HORIZON.SURVEY_FREQ.FCT_SOURCE`. Codelist values са discoverable чрез `/datastructure/ECB/<DSD_id>?references=children`

5. **Quarterly forward-fill за SPF в macro_vector:** ECB SPF е quarterly, други dims са monthly. `resample("MS").ffill()` aligns.

6. **`is_rate` flag в SERIES dict** — определя дали YoY column показва pp delta. Display layer (Phase 7) добави transform-aware override чрез `change_kind()`.

7. **`set` vs `dict` literal** — `BPS_SIDS_OVERRIDE: set[str] = {}` е dict (празен литерал)! Винаги `set()` за празен set.
